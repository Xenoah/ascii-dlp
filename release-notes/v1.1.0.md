# ascii-dlp v1.1.0

動画をターミナル上のASCIIアートとして音声付き再生するCLIです。このリリースから、FFmpeg・ffprobe・yt-dlp・音声ライブラリを1回の`pip install`でまとめて導入できます。FFmpegの手動インストールやPATH設定は不要です。

## インストール / 更新

Python 3.10以上が必要です。FFmpegを含む依存関係も取得するため、初回は大きめのダウンロードになります。

### Windows（コマンドプロンプト / PowerShell）

```powershell
py -m pip install --upgrade "https://github.com/Xenoah/ascii-dlp/releases/download/v1.1.0/ascii_dlp-1.1.0-py3-none-any.whl"
py -m ascii_dlp doctor
```

`py`コマンドがない環境では、`py`を`python`または`python3`へ置き換えてください。

### macOS / Linux

```bash
python3 -m pip install --upgrade "https://github.com/Xenoah/ascii-dlp/releases/download/v1.1.0/ascii_dlp-1.1.0-py3-none-any.whl"
python3 -m ascii_dlp doctor
```

## 使い方

動画URLを引用符で囲んで指定します。`動画URL`という文字列は例なので、実際のURLへ置き換えてください。

```bash
ascii-dlp "https://www.youtube.com/watch?v=VIDEO_ID"
```

コマンドがPATHから見つからない場合は、次の形式でも実行できます。

```bash
python3 -m ascii_dlp "https://www.youtube.com/watch?v=VIDEO_ID"
```

Windowsで`py`を使う場合:

```powershell
py -m ascii_dlp "https://www.youtube.com/watch?v=VIDEO_ID"
```

ローカル動画を再生:

```powershell
py -m ascii_dlp "C:\Videos\sample.mp4"
```

描画を軽くする:

```bash
ascii-dlp "動画URL" --fps 10 --max-width 80
```

無音で再生:

```bash
ascii-dlp "動画URL" --no-audio
```

先にダウンロードして安定再生:

```bash
ascii-dlp "動画URL" --download
```

ログイン済みブラウザのCookieを利用:

```bash
ascii-dlp "動画URL" --cookies-from-browser edge
```

操作キー:

- `Space`: 一時停止 / 再開
- `←` / `→`: 5秒戻る / 進む
- `↓` / `↑`: 30秒戻る / 進む
- `Q`: 終了

すべてのオプションは`ascii-dlp --help`、依存関係の状態は`ascii-dlp doctor`で確認できます。

## v1.1.0の主な変更

- OS別のFFmpeg・ffprobeバイナリを依存関係として自動導入
- `ffplay`を廃止し、クロスプラットフォームの`miniaudio`で音声を再生
- 外部FFmpegがPATHにある場合は従来どおり優先して利用
- `--ffmpeg-location`による外部FFmpegの明示指定も継続サポート
- `doctor`に内蔵/外部FFmpegの識別とminiaudio診断を追加
- 音声デバイスを開始できない場合は、映像を止めずに無音再生へフォールバック

動画の利用規約、著作権、アクセス権を守り、視聴・取得する権限があるコンテンツにだけ使用してください。
