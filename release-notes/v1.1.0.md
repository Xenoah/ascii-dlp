# ascii-dlp v1.1.0

必要な実行環境をコマンドラインだけで導入できるようにしたリリースです。

## 主な変更

- `pip install`時にOS別のFFmpeg・ffprobeバイナリを依存関係として自動導入
- `ffplay`を廃止し、クロスプラットフォームの`miniaudio`で音声を再生
- 外部FFmpegがPATHにある場合は従来どおり優先して利用
- `--ffmpeg-location`による外部FFmpegの明示指定も継続サポート
- `ascii-dlp doctor`に内蔵/外部FFmpegの識別とminiaudio診断を追加
- 音声デバイスを開始できない環境では、映像を止めずに無音再生へフォールバック

## インストール / 更新

Windows:

```powershell
py -m pip install --upgrade "https://github.com/Xenoah/ascii-dlp/releases/download/v1.1.0/ascii_dlp-1.1.0-py3-none-any.whl"
```

macOS / Linux:

```bash
python3 -m pip install --upgrade "https://github.com/Xenoah/ascii-dlp/releases/download/v1.1.0/ascii_dlp-1.1.0-py3-none-any.whl"
```

インストール後に`python -m ascii_dlp doctor`（Windowsでは`py -m ascii_dlp doctor`）で環境を確認できます。

動画の利用規約、著作権、アクセス権を守り、視聴・取得する権限があるコンテンツにだけ使用してください。
