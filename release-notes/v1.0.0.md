# ascii-dlp v1.0.0

最初の安定版リリースです。`yt-dlp` と FFmpeg を利用し、動画URLまたはローカル動画をターミナル上のASCIIアートとして音声付きで再生できます。

## 主な機能

- `yt-dlp` 対応サイトの動画URLとローカル動画ファイルを再生
- 端末サイズと動画の縦横比に合わせたASCII描画
- 音声再生、一時停止、前後シーク
- FPS、最大サイズ、文字セット、ガンマ、コントラスト、明暗反転の調整
- 分離ストリーム、Cookie利用、直接再生失敗時の一時ダウンロード・結合
- Windows、macOS、LinuxとPython 3.10以降に対応
- `doctor`によるyt-dlp・FFmpeg環境の診断

## 必要なもの

- Python 3.10以上
- FFmpeg（`ffmpeg`、`ffprobe`。音声には`ffplay`）

## 配布物

- Python wheel（`.whl`）
- ソースアーカイブ（`.tar.gz`）

動画の利用規約、著作権、アクセス権を守り、視聴・取得する権限があるコンテンツにだけ使用してください。
